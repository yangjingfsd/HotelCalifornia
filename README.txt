This project is based on Laravel, composer, and bootstrap.

HASH password is implemented.

Weather API is used to obtain weather data, and the data is shown
in the home page.

Cookie is used to track the login status of user.

After login, menu related to user show up, and login menu disappear.
otherwise, it is the reverse.

After login, the user's information can be extracted from database
automatically when making reservation. 

The hotel room information is extracted from a text file named hotel.txt,
which has already been formated as HTML style.

SEO Compatibility is addressed in the robots.txt file.
